# Character-Cookbook
Création et gestion de personnages de jeu de rôle

 # Informations importantes !

 Ce projet utilise la fonctionnalité sub-module de GitHub (pour plus de flexibilité). Pour cloner ce projet et pouvoir vous en servir sur votre propre machine, vous devez lancer la commande suivante :

```sh
git clone --recursive <repo-url> 
```
(je suppose que) : <repo-url> = LemanTheo/Character-Cookbook 

Si vous  clonez simplement ce projet vous devrez ensuite effectuer la commande suivante :

```sh
git submodule update --init --recursive
```
