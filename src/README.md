# Character Cookbook

In order to correctly *cook* our characters, we need organisation.

- The Bookshelf folder holds our cookbooks, the "libraries" (in a way) that will be used to store the content that we will add to the character sheets.
- The Libraries folder has the cooking utensils, the classes & functions that the application holds and uses (at least, those we created).
